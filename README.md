# Slide Strategy Game

A real-time multiplayer slide strategy game built with React, Vite, Express, and Socket.IO.

## Prerequisites

- Node.js

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

This will run both the frontend Vite build and the backend Express server concurrently.

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm start`: Starts the application in production mode.
- `npm run preview`: Previews the production build.
